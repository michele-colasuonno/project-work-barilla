# Project Work – Report di Sostenibilità Barilla

Questo progetto consiste nello sviluppo di una semplice pagina web in HTML e CSS
finalizzata alla divulgazione delle attività di sostenibilità del Gruppo Barilla
e al download dei relativi report di sostenibilità.

## Tecnologie utilizzate
- HTML5
- CSS3

## Obiettivo
Creare una pagina web intuitiva e accessibile che permetta agli utenti
di informarsi sulle politiche di sostenibilità dell’azienda e di scaricare
i report ufficiali pubblicati.
